#################
Video API
#################

==========
sections
==========


:kbd:`/rest/video/sections?v=2`

Returns a list of video feeds

Sample *response* ::

    [
        {
            "value": "ted", 
            "title": "TED Talks"
        }, 
        {
            "value": "vimeo", 
            "title": "Vimeo Staff Picks"
        }
    ]
