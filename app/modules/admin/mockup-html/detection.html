Device Detection

The device detection service recognizes and categorizes each user's web browser so that the appropriate class of web UI can be delivered. You can run this service on your own web server, or use an externally hosted one such as the reference service provided by Modo Labs.

  • Version of the Kurogo device detection service
  • Use the external version of the Kurogo device detection service hosted by Modo Labs. This introduces a new external dependecy for your website but ensures that your system will always be accessing the latest device detection database.
  • Path to the mobi-service device database file within your site
  • URL of the externally hosted device-detection service
  • seconds Time (in seconds) that the device detection service should be cached on the server
